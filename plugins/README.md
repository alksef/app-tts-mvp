# File Logger Plugin

Тестовый плагин для app-tts, который записывает полученные тексты в файл.

## Установка

Скопируйте `file_logger_plugin.dll` в папку `plugins/` рядом с `app-tts.exe`.

## Конфигурация

В настройках приложения (вкладка Plugins) укажите путь к файлу лога:

- **File Path**: `tts-log.txt` (относительный путь)

Файл будет создаваться в директории приложения.

## Формат лога

```
[2025-01-31 12:34:56]
Первое сообщение

------
[2025-01-31 12:35:10]
Второе сообщение

------
```

## Сборка из исходников

```bash
cd plugins
cargo build --release
```

DLL будет в `target/release/file_logger_plugin.dll`.
